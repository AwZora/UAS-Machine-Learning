## Project Overview

### 1. Fraud Detection

Objective:

Predict whether an online transaction is fraudulent using transaction data.

Main Components:

- Data preprocessing
- Feature engineering
- Class imbalance handling
- Random Forest Classifier
- Hyperparameter tuning using Optuna
- Model evaluation
- MLflow tracking

Dataset:

- train_transaction.csv

Folder:

```text
Fraud-Detection/
```

---

### 2. Regression Prediction

Objective:

Predict the release year of a song using numerical audio features.

Main Components:

- Data preprocessing
- Feature scaling
- Random Forest Regressor
- Hyperparameter tuning using Optuna
- Model evaluation
- MLflow tracking
- LIME model interpretation

Dataset:

- midterm-regresi-dataset.csv

Folder:

```text
Regression-Prediction/
```
