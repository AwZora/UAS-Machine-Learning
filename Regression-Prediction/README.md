# Regression Prediction Project

## Project Purpose

This project aims to build an end-to-end regression pipeline to predict a continuous target value using machine learning techniques. The objective is to predict the release year of a song based on its audio features.

## Dataset

Dataset: midterm-regresi-dataset.csv

- Total samples: 515,345
- Total features: 90
- Target variable: Release Year (first column)

## Workflow

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Scaling using StandardScaler
5. Train-Test Split (80:20)
6. Model Training
7. Hyperparameter Tuning using Optuna
8. Model Evaluation
9. MLflow Tracking
10. Model Interpretation using LIME

## Model Used

- Random Forest Regressor

## Evaluation Results

| Metric | Value |
|----------|----------|
| MAE | 6.6088 |
| RMSE | 9.4040 |
| R² Score | 0.2569 |

### Interpretation

- MAE of 6.61 indicates that the prediction differs from the actual release year by approximately 6.6 years on average.
- RMSE of 9.40 shows the magnitude of prediction error.
- R² Score of 0.2569 means the model explains approximately 25.69% of the variance in the target variable.

## LIME Interpretation

LIME (Local Interpretable Model-Agnostic Explanations) was used to explain individual predictions generated by the Random Forest Regression model.

The explanation identified the most influential features contributing positively and negatively to the predicted release year. This improves model transparency and helps understand how predictions are generated.
